DEPENDENCIES:
- Numpy
- Pandas

RUNNING:
- assuming all dependencies are installed the script can be run as any python script
- {/path/to/python} main.py

INPUT:
- input files are coded into the file itself
- can be changed by changing the filepaths to these data files
- input file must be in the following format (feature1, feature2, feature3, feature4, class-x) where x is an integer and features are floats

OUTPUT:
- all output is printed into the console
- output is saved in data.out file in current directory
